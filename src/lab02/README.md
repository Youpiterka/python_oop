# ЛР-2 — Коллекция объектов

## Описание

Реализован контейнерный класс `CharacterCollection` для хранения объектов `Character` из ЛР-1.

Предметная область: **Персонажи игры**

## Структура файлов

```
lab02/
├── model.py        — класс Character (из лр1)
├── collection.py   — класс CharacterCollection (контейнер)
├── demo.py         — демонстрация работы
└── README.md       — этот файл
```

## Что реализовано

- `add(item)` — добавить персонажа (с проверкой типа)
- `remove(item)` — удалить персонажа
- `get_all()` — получить список всех персонажей
- `find_by_name(name)` — поиск по имени
- `find_by_level(level)` — поиск по уровню
- `__len__()` — поддержка `len(col)`
- `__iter__()` — поддержка `for char in col`
- Защита от дубликатов по имени
- `__getitem__(index)` — индексация `col[0]`
- `remove_at(index)` — удаление по индексу
- `sort_by_name()` — сортировка по имени
- `sort_by_level()` — сортировка по уровню
- `sort_by_health()` — сортировка по здоровью
- `sort(key)` — универсальная сортировка
- `get_alive()` — только живые → новая коллекция
- `get_dead()` — только мёртвые → новая коллекция
- `get_high_level(min_level)` — высокоуровневые → новая коллекция
