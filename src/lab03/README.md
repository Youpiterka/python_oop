ЛР-3 — Наследование и иерархия классов

## 1. Цель работы
Изучить механизм наследования классов в Python и научиться создавать иерархию объектов с использованием переопределения методов наследования и полиморфизма.

---

## 2. Краткое описание

В работе реализован базовый класс `Character` и два наследника: `Warrior` и `Mage`.  
Классы расширяют базовый функционал, добавляя уникальные характеристики и поведение.  
Также реализован полиморфизм через метод `calculate_power()`.

---

## 3. Используемые новые методы

### Warrior
- `strong_hit()` — усиленная атака (удар с увеличенным уроном)
- `calculate_power()` — расчёт силы с учётом брони и выносливости
- `__str__()` — расширенный вывод информации

### Mage
- `magic_hit()` — магическая атака с учётом интеллекта
- `calculate_power()` — расчёт силы с учётом маны и интеллекта
- `__str__()` — расширенный вывод информации

---

## 4. Демонстрация работы

В `demo.py`:

- создаются объекты `Warrior` и `Mage`
- выводится информация о персонажах
- вызываются методы `strong_hit()` и `magic_hit()`
- демонстрируется полиморфизм через `calculate_power()`
- используется коллекция с фильтрацией по типу объектов
